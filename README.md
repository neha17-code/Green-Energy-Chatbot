# 🌱 Green Energy Chatbot

## 📌 Project Overview
The Green Energy Chatbot is an AI-based application that predicts a sustainability score based on energy usage, raw material consumption, waste generation, and other environmental factors.

---

## 🚀 Features
- Sustainability score prediction
- Machine learning based model
- Streamlit web interface
- Real-time results
- User-friendly design

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- Joblib
